Census Workplace Data Pack - 2011 - Local Authority District Geodata Pack: Birmingham (E08000025)


+ Abstract
This geodata pack provides the Census Workplace Data Pack for the year 2011 covering the Local Authority District: Birmingham (E08000025)


+ Contents
	 - readme.txt: Information about the CDRC Geodata pack
	 - metadata.xml: Metadata
	 - tables: Folder containing the csv files
	 - shapefiles: Folder containing the shapefiles
